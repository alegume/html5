Layouts

O HTML 5 traz novas tags de caráter semântico para compor nossos sites.

Principais tags

Explicação

  • <header> - Define um cabeçalho para um documento ou uma
  • <nav> - Define um recipiente para menu de navegação
  • <section> - Define uma seção
  • <article> - Define um artigo independente
  • <main> - Define a seção principal da página
  • <aside> - Define o conteúdo lateral
  • <footer> - Define um rodapé de um documento ou de uma seção

Trê formas de trabalhar com layout

  • CSS float property
  • CSS flexbox
  • CSS framework
  • CSS Grid Layout Module
